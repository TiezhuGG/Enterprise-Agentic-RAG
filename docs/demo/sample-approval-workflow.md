# 权限与审批流程说明

## 部门审批

普通费用申请由直属部门负责人初审。初审重点包括事项真实性、预算归属和材料完整性。

涉及跨部门预算的申请，应由相关部门负责人共同确认后再进入财务复核。

## 金额分级

- 单笔 3000 元以内的费用，由部门负责人审批。
- 单笔 3000 元至 10000 元的费用，由部门负责人审批后提交财务部复核。
- 单笔超过 10000 元的费用，需要财务部复核，并由总经理最终批准。

## 密级访问策略

PUBLIC 资料可被已登录用户在授权空间内读取。

INTERNAL 资料要求用户属于相同租户，并且至少拥有该知识空间的 VIEWER 身份。

CONFIDENTIAL 资料默认仅允许空间 OWNER、被授权部门或具备 `knowledge.confidential.read` 权限的用户读取。

## 异常处理

如果审批材料不完整，财务部应退回申请并说明缺失项。员工应在 3 个工作日内补充材料。

如果发现重复报销、虚假票据或跨租户误共享，应立即暂停流程并通知空间 OWNER 进行权限复核。
